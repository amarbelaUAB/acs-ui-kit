AI Setup Guide

Set Up an ASP.NET MVC App with the ACS / UAB UI Kit

Use this guide to create the app, add the required folders, and give Copilot the correct instructions before applying the ACS / UAB UI Kit.

Usage Instructions

Follow these three steps when starting a new ASP.NET MVC or ASP.NET Core MVC application.

Step 1

Create the app

Create the ASP.NET MVC or ASP.NET Core MVC app normally in Visual Studio. Run it once before applying the UI Kit.

Step 2

Add the folders

Add .github/copilot-instructions.md and the acs-ui-kit-template folder in the solution root.

Step 3

Ask Copilot for a plan

Use Agent mode and ask Copilot to inspect the app before making changes.

Folder Structure

Keep the UI Kit resources separate from the MVC project files, but inside the same solution workspace.

Solution root 

MyNewApp/
├── .github/
│   └── copilot-instructions.md
├── acs-ui-kit-template/
